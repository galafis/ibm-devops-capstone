Visão Geral do Projeto

Este projeto representa o trabalho final do IBM DevOps & Software Engineering Professional Certificate. Ele demonstra competências avançadas em DevOps, integração contínua (CI), entrega contínua (CD) e engenharia de software. A plataforma desenvolvida oferece uma solução completa com um pipeline CI/CD robusto, containerização e automação de deployment, focando na entrega de valor através de uma aplicação de análise de dados interativa.

A aplicação principal é um dashboard interativo construído com Streamlit, que permite a visualização e interação com dados, análises avançadas e demonstração de capacidades de Machine Learning e Analytics Engine.

Características Principais

Dashboard Interativo

Interface web responsiva e intuitiva para visualização de dados.

Processamento de Dados

Pipeline robusto e escalável para manipulação eficiente de informações.

Analytics Avançado

Análises estatísticas e modelos de Machine Learning para insights valiosos.

API RESTful

Endpoints para integração com sistemas externos.

Segurança e Compliance

Autenticação segura, controle de acesso e trilha de auditoria.

CI/CD Automatizado

Demonstração de pipeline de integração e entrega contínua.

Demonstração da Aplicação

Abaixo, você pode ver uma prévia do dashboard interativo. Para uma experiência completa, clone o repositório e execute a aplicação localmente.

A imagem acima é um placeholder. Em um ambiente real, seria substituída por um screenshot ou GIF animado do dashboard.

Para executar a aplicação:

git clone https://github.com/galafis/ibm-devops-capstone.git
cd ibm-devops-capstone
pip install -r requirements.txt
python src/main_platform.py

Então, acesse http://localhost:8501 no seu navegador.

Tecnologias Utilizadas

Este projeto utiliza uma variedade de tecnologias modernas, incluindo Python, Streamlit, Pandas, Plotly, Scikit-learn, SQLite, Docker, Kubernetes e Jenkins (conceitual).
